E-commerce Web Application
A functional e-commerce frontend built with React and Redux, featuring a dynamic product catalog, shopping cart logic, and a modular checkout system.

Key Features
State Management: Handled via Redux Toolkit for cart operations (add/remove/update).

Responsive UI: Built using CSS Modules for scoped and maintainable styling.

Checkout Flow: Includes a dedicated checkout page with shipping forms and an order summary.

Payment Ready: Structured to integrate with the Razorpay payment gateway.

Tech Stack
Frontend: React.js

State: Redux / Redux Toolkit

Styling: CSS Modules (CSS3)

Icons/Assets: FontAwesome / Google Fonts

Getting Started
Clone the repository: git clone https://github.com/mownika01/Ecommerce.git

Install dependencies: npm install

Run the project: npm start
